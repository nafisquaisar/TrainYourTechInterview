🔄 Android Adapter - Complete Notes

📘 What is an Adapter?

An Adapter acts as a bridge between a data source and UI components like ListView, RecyclerView, or Spinner. It prepares data and binds it to the layout views efficiently.

🧩 Types of Adapters

1. ArrayAdapter 📋

  • Used With: ListView, Spinner

  • Purpose: To display simple data like strings or integers.

  • Example:

    val names = arrayOf("John", "Emma", "Liam")
val adapter = ArrayAdapter(this, android.R.layout.simple_list_item_1, names)
listView.adapter = adapter

2. CursorAdapter 🧠

  • Used With: ListView

  • Purpose: To bind data from a SQLite database or Content Provider.

  • Example:

    val cursor = db.query("students", null, null, null, null, null, null)
val adapter = SimpleCursorAdapter(
    this,
    android.R.layout.simple_list_item_2,
    cursor,
    arrayOf("name", "age"),
    intArrayOf(android.R.id.text1, android.R.id.text2),
    0
)
listView.adapter = adapter

3. RecyclerView.Adapter 🧱

  • Used With: RecyclerView

  • Purpose: To handle complex, large, and dynamic data sets efficiently.

  • Example:

    class UserAdapter(private val userList: List<String>) : RecyclerView.Adapter<UserAdapter.ViewHolder>() {

    class ViewHolder(val view: View) : RecyclerView.ViewHolder(view) {
        val textView: TextView = view.findViewById(R.id.textView)
    }

    override fun onCreateViewHolder(parent: ViewGroup, viewType: Int): ViewHolder {
        val view = LayoutInflater.from(parent.context).inflate(R.layout.item_user, parent, false)
        return ViewHolder(view)
    }

    override fun onBindViewHolder(holder: ViewHolder, position: Int) {
        holder.textView.text = userList[position]
    }

    override fun getItemCount() = userList.size
}

    Set Adapter:

    val recyclerView: RecyclerView = findViewById(R.id.recyclerView)
recyclerView.layoutManager = LinearLayoutManager(this)
recyclerView.adapter = UserAdapter(listOf("Ava", "Noah", "Sophia"))

4. ListAdapter ⚡

  • Used With: RecyclerView

  • Purpose: A subclass of RecyclerView.Adapter that automatically handles list updates efficiently using DiffUtil.

  • Why Use It: Reduces boilerplate code for handling changes in datasets.

  • Example:

    class UserListAdapter : ListAdapter<String, UserListAdapter.ViewHolder>(DiffCallback()) {

    class ViewHolder(val view: View) : RecyclerView.ViewHolder(view) {
        val textView: TextView = view.findViewById(R.id.textView)
    }

    class DiffCallback : DiffUtil.ItemCallback<String>() {
        override fun areItemsTheSame(oldItem: String, newItem: String) = oldItem == newItem
        override fun areContentsTheSame(oldItem: String, newItem: String) = oldItem == newItem
    }

    override fun onCreateViewHolder(parent: ViewGroup, viewType: Int): ViewHolder {
        val view = LayoutInflater.from(parent.context).inflate(R.layout.item_user, parent, false)
        return ViewHolder(view)
    }

    override fun onBindViewHolder(holder: ViewHolder, position: Int) {
        holder.textView.text = getItem(position)
    }
}

    Set Adapter:

    val adapter = UserListAdapter()
recyclerView.adapter = adapter
adapter.submitList(listOf("Ava", "Noah", "Sophia"))

📋 What is RecyclerView?

RecyclerView is an advanced and flexible version of ListView used to display large or dynamic data efficiently. It recycles item views that scroll off-screen, improving memory and performance.

Why We Use RecyclerView:

  • Handles large data efficiently.
  • Provides flexible layout management.
  • Supports animations and item decorations.

🧱 Types of RecyclerView Layouts

LayoutManager Type Description Example
LinearLayoutManager Displays items in a vertical or horizontal list. Chat app, contact list
GridLayoutManager Displays items in a grid format. Gallery, product grid
StaggeredGridLayoutManager Displays items in a staggered grid. Pinterest-style layout

🎯 UI Components That Use Adapters

UI Component Description Common Adapter Types
ListView Displays a vertical scrollable list. ArrayAdapter, CursorAdapter, BaseAdapter
RecyclerView Advanced list with better performance and flexibility. RecyclerView.Adapter, ListAdapter
Spinner Displays a dropdown list for selection. ArrayAdapter, BaseAdapter

💡 Key Points

  • Adapters connect data sources (arrays, databases, lists) to UI elements.
  • They improve performance by recycling views instead of recreating them.
  • Choose adapter type based on data complexity and UI component.
