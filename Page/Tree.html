🌳 Tree Data Structure – Complete Notes

1️⃣ What is a Tree?

A tree is a non-linear data structure that represents data in a hierarchical format.

  • It consists of nodes connected by edges
  • The topmost node is called the root
  • Each node can have zero or more child nodes

👉 Real-life Examples

  • File system (folders & files)
  • Organization hierarchy
  • HTML DOM structure

2️⃣ Tree Structure (Visual Representation) 

            A
           / \
          B   C
         / \   \
        D   E   F

Explanation

  • A is the root node
  • B and C are children of A
  • D and E are children of B
  • F is child of C
  • D, E, F are leaf nodes

3️⃣ Basic Tree Terminology

Term Description
Node An element of a tree
Root Topmost node
Parent Immediate predecessor
Child Immediate successor
Leaf Node with no children
Edge Connection between nodes
Height Longest path from node to leaf
Depth Distance from root
Subtree Tree formed from any node

3️⃣ Properties of Tree

  • Only one root node
  • No cycles
  • A tree with n nodes has n-1 edges
  • Every child has exactly one parent

4️⃣ Types of Tree

🔹 General Tree

  • A node can have any number of children

🔹 Binary Tree

  • Each node has at most 2 children (left & right)

🔹 Binary Search Tree (BST)

  • Left subtree < root < right subtree

🔹 Full Binary Tree

  • Every node has either 0 or 2 children

🔹 Complete Binary Tree

  • All levels filled except last (filled left to right)

🔹 Balanced Tree

  • Height difference of subtrees is limited

5️⃣ Binary Tree Node Structure (Java) 

class Node {
    int data;
    Node left, right;

    Node(int data) {
        this.data = data;
        left = right = null;
    }
}

6️⃣ Tree Traversals

Traversal means visiting all nodes of a tree.

🔹 Depth First Traversal (DFS)

1️⃣ Inorder (Left → Root → Right)

  • In inorder traversal, the left subtree is visited first, then the root node, and finally the right subtree.
void inorder(Node root) {
    if (root == null) return;
    inorder(root.left);
    System.out.print(root.data + " ");
    inorder(root.right);
}

2️⃣ Preorder (Root → Left → Right)

  • In preorder traversal, the root node is visited first, followed by the left subtree, then the right subtree.
    It is often used to copy a tree or generate a prefix expression.
void preorder(Node root) {
    if (root == null) return;
    System.out.print(root.data + " ");
    preorder(root.left);
    preorder(root.right);
}

3️⃣ Postorder (Left → Right → Root)

  • In postorder traversal, the left subtree is visited first, then the right subtree, and the root node is visited last.
    It is useful for deleting a tree or evaluating postfix expressions.
void postorder(Node root) {
    if (root == null) return;
    postorder(root.left);
    postorder(root.right);
    System.out.print(root.data + " ");
}

🔹 Breadth First Traversal (Level Order) 

import java.util.*;

void levelOrder(Node root) {
    if (root == null) return;
    Queue<Node> q = new LinkedList<>();
    q.add(root);

    while (!q.isEmpty()) {
        Node curr = q.poll();
        System.out.print(curr.data + " ");
        if (curr.left != null) q.add(curr.left);
        if (curr.right != null) q.add(curr.right);
    }
}

7️⃣ Binary Search Tree (BST)

📌 Property

  • Left subtree contains smaller values
  • Right subtree contains greater values

📌 Insertion in BST 

Node insert(Node root, int value) {
    if (root == null) return new Node(value);

    if (value < root.data)
        root.left = insert(root.left, value);
    else
        root.right = insert(root.right, value);

    return root;
}

8️⃣ Applications of Tree

  • Searching (BST)
  • Sorting (Heap)
  • Expression evaluation
  • Routing algorithms
  • Database indexing (B-Trees)
