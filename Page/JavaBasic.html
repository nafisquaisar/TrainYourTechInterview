Java Interview Questions – One-Sentence Answers

Core Java

  1. 📘 What is Java?
    Java is a high-level, object-oriented programming language used to build platform-independent applications.

  2. 🌍 Why is Java platform independent?
    Java is platform independent because its compiled bytecode runs on the JVM, which is available on different operating systems.

  3. 🧰 Difference between JDK, JRE, and JVM?
    JDK is used to develop Java programs, JRE provides the runtime environment, and JVM executes the bytecode.

  4. ⭐ What are the main features of Java?
    Java is object-oriented, platform independent, secure, robust, portable, and supports multithreading.

  5. 🧠 What is OOP in Java?
    OOP is a programming approach that uses objects and classes to build reusable, secure, and maintainable software

  6. 🧱 What are the four pillars of OOP?
    The four pillars of OOP are Encapsulation, Inheritance, Polymorphism, and Abstraction.

  7. 🏗️ Difference between class and object?
    A class is a blueprint, while an object is a real-world instance of that class.

  8. 👨‍👦 What is inheritance?
    Inheritance means that a child class inherits the properties and methods of a parent class.

  9. 🎭 What is polymorphism?
    Polymorphism allows the same method to behave differently based on the object.

  10. ⏱️ Difference between compile-time and runtime polymorphism?
    Compile-time polymorphism is achieved using method overloading, while runtime polymorphism is achieved using method overriding.

  11. ➕ What is method overloading?
    Method overloading means a class has multiple methods with the same name but different parameters.

  12. 🔁 What is method overriding?
    Method overriding is a runtime polymorphism concept where a child class provides its own implementation of a method defined in the parent class.

  13. 🔒 What is encapsulation?
    Encapsulation is the process of wrapping data and methods into a single unit and restricting access using access modifiers.

  14. 🎯 What is abstraction?
    Abstraction is an OOP concept that hides implementation details and shows only the essential features to the user.

  15. 🔀 Difference between abstract class and interface?
    An abstract class can have both abstract and concrete methods.
    An interface contains only abstract methods (before Java 8).
    A class extends an abstract class but implements an interface.

  16. 🚫 Can we create an object of an abstract class?
    No, we cannot directly create an object of an abstract class because it is incomplete and meant to be inherited.

  17. 🛂 What are access modifiers in Java?
    Access modifiers define the visibility and accessibility of classes, methods, and variables.

  18. 🧩 Difference between public, private, protected, and default?

    public → accessible everywhere

    private → accessible within the same class

    protected → accessible in same package or subclass

    default → accessible within the same package

  19. 🧷 What is the static keyword?
    The static keyword is used to create class-level members that are shared among all objects.

  20. 🛑 What is the final keyword?
    The final keyword restricts modification by preventing reassignment, overriding, or inheritance.

  21. 🧪 Difference between final, finally, and finalize?
    final is used to restrict modification, finally is used to execute cleanup code in a try-catch block, and finalize is called by the garbage collector before an object is destroyed

  22. 🏗️ What is a constructor?
    A constructor is a special method used to initialize objects when they are created.It has the same name as the class and no return type.

  23. 🧩 Types of constructors?
    Java supports default constructors and parameterized constructors.

  24. 👉 What is the this keyword?
    The this keyword refers to the current object of a class.

  25. 👆 What is the super keyword?
    The super keyword refers to the parent class object.

  26. 🚨 What is exception handling?
    Exception handling is used to manage runtime errors and prevent abnormal program termination.

  27. ⚠️ Difference between checked and unchecked exceptions?
    Checked exceptions are verified at compile time(IOException), while unchecked exceptions occur at runtime(NullPointerException).

  28. 🛠️ What is try-catch-finally?
    Try-catch-finally is used to handle exceptions and execute cleanup code regardless of errors.
    try contains risky code, catch handles the exception, and finally executes always.

  29. ❌ What is NullPointerException?
    NullPointerException occurs when an application tries to access an object reference that is null.

  30. 📊 Difference between Array and ArrayList?
    Array has a fixed sizeand can store primitives and objects.
    , while ArrayList supports dynamic resizing and stores only objects..

Collections & Memory

  1. 📦 What is the Collection Framework?
    The Collection Framework provides predefined classes and interfaces to store and manipulate data efficiently.

  2. 📋 Difference between List, Set, and Map?
    List allows duplicates and maintains order, Set stores unique elements, and Map stores data as key-value pairs.

  3. 🔗 Difference between ArrayList and LinkedList?
    ArrayList uses contiguous memory and provides fast access, while LinkedList uses nodes and allows faster insertion and deletion.

  4. 🗂️ Difference between HashMap and HashSet?
    HashMap stores key-value pairs, while HashSet stores only unique values.

  5. 🔐 Difference between HashMap and Hashtable?
    HashMap is not synchronized and allows null keys, while Hashtable is synchronized and does not allow null keys.

  6. 🔄 What is an Iterator?
    Iterator is used to traverse elements of a collection one by one.

  7. 🧹 What is Garbage Collection?
    Garbage Collection automatically removes objects that are no longer referenced in memory.

  8. 🧠 Difference between heap and stack memory?
    Heap memory stores objects, while stack memory stores method calls and local variables.

  9. 🧵 What is String Pool?
    String Pool is a special memory area used to store string literals to optimize memory usage.

  10. ✍️ Difference between String, StringBuilder, and StringBuffer?
    String is immutable,
    StringBuilder is mutable and faster,
    and StringBuffer is mutable and thread-safe.
