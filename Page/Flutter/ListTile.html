↕️ListTile

Creates a fixed-height row that usually contains text, icons, or widgets in a list.

ListTile(
  leading: Icon(Icons.person),
  title: Text('Nafis Quaisar'),
  subtitle: Text('Android Developer'),
  trailing: Icon(Icons.arrow_forward),
);

Properties of ListTile

1. leading

  • Left side widget.
  • Mostly icon or avatar.

2. title

  • Main text of the tile.

3. subtitle

  • Secondary text. Shown below title.

4. trailing

  • Right side widget.
  • Icon, switch, button, etc.

5. onTap

  • Tile tap event.
  • Navigation ya action ke liye.

6. onLongPress

  • Long press action.

7. enabled

  • True/false. Disabled tile tap nahi hoga.

8. selected

  • True ho to tile highlighted dikhega.

9. dense

  • True ho to compact layout banata hai.

10. contentPadding

  • Andar ka padding control karta hai.

11. isThreeLine

  • True ho to title + subtitle + extra line allow.

12. visualDensity

  • Tile ki spacing tight ya loose kar sakte ho.

13. shape

  • Custom border, rounded corners.

14. tileColor

  • Background color.

15. selectedTileColor

  • When tile is selected.

16. hoverColor

  • Web hover effect.

17. focusColor

  • Keyboard focus effect.

18. minLeadingWidth

  • Leading widget ka minimum width.

Example with all major properties 

ListTile(
  leading: CircleAvatar(child: Icon(Icons.person)),
  title: Text('Nafis'),
  subtitle: Text('Flutter Developer'),
  trailing: Icon(Icons.arrow_forward_ios),
  onTap: () {},
  dense: false,
  selected: false,
  tileColor: Colors.white,
  contentPadding: EdgeInsets.symmetric(horizontal: 16),
);

ListWheelScrollView & ClipRRect (Flutter Notes)

1. ListWheelScrollView

What is ListWheelScrollView?

ListWheelScrollView is a Flutter widget that displays its children in a cylindrical wheel style, similar to iOS picker wheels.

It is commonly used for:

  • 🔢 Number pickers
  • 🎂 Age selectors
  • 📅 Date / time pickers
  • 🍎 iOS-style selection UI

Key Features

  • ↕️ Vertical scrolling only
  • 🌀 3D wheel (curved) effect
  • 🎯 Center item appears highlighted
  • 🧲 Smooth snapping scrolling

Basic Example 

ListWheelScrollView(
  itemExtent: 60,
  children: const [
    Text("One"),
    Text("Two"),
    Text("Three"),
    Text("Four"),
  ],
);

Important Properties

1. itemExtent (required)

Height of each item in the wheel.

itemExtent: 60,

2. diameterRatio

Controls the curvature of the wheel.

diameterRatio: 1.5,

Smaller value → more curved wheel.

3. perspective

Controls 3D depth effect.

perspective: 0.003,

4. useMagnifier & magnification

Magnifies the center item.

useMagnifier: true,
magnification: 1.2,

5. physics

Controls scrolling behavior.

physics: FixedExtentScrollPhysics(),

Optimized Example (Builder version) 

int selectedIndex = 0;

ListWheelScrollView.useDelegate(
  itemExtent: 60,
  physics: const FixedExtentScrollPhysics(),
  onSelectedItemChanged: (index) {
    selectedIndex = index;
  },
  childDelegate: ListWheelChildBuilderDelegate(
    childCount: 10,
    builder: (context, index) {
      return Center(
        child: Text(
          "Item $index",
          style: const TextStyle(fontSize: 22),
        ),
      );
    },
  ),
);

When to Use ListWheelScrollView

  • 🔢 Number pickers
  • 🎂 Age selection
  • ⏰ Time and date pickers
  • 🍎 iOS-style selector UI

ListWheelScrollView vs ListView

Feature ListView ListWheelScrollView
Layout Flat list Curved wheel
Direction Vertical / Horizontal Vertical only
UI Style Normal list iOS picker
Snapping Manual Built-in

2. ClipRRect

What is ClipRRect?

ClipRRect is used to clip a widget into a rounded rectangle shape.

It is commonly used to:

  • 🖼️ Round image corners
  • 🃏 Clip cards
  • 🔵 Create circular or rounded UI elements

Basic Example 

ClipRRect(
  borderRadius: BorderRadius.circular(20),
  child: Image.asset(
    'assets/images/img.jpg',
    width: 200,
    height: 200,
    fit: BoxFit.cover,
  ),
);

Common Use Cases

  • 👤 Rounded profile images
  • 🖼️ Image cards
  • ✨ UI with smooth corners

ClipRRect vs Container(borderRadius)

Feature ClipRRect Container(borderRadius)
Clips child Yes No
Works for images Yes Sometimes
Performance Slight cost Better

Important Note

Container with borderRadius does not clip its child unless used with ClipRRect.

Interview One-Liners

  • 🎡 ListWheelScrollView creates a cylindrical scrolling list used in picker UIs.
  • 📏 itemExtent defines the height of each wheel item.
  • ✂️ ClipRRect clips its child widget into a rounded rectangle.
  • ⚠️ Container borderRadius does not clip children by default.

Summary

  • 🎡 Use ListWheelScrollView for wheel-style selectors
  • ✂️ Use ClipRRect to clip widgets with rounded corners
  • ✅ Both are commonly used in modern Flutter UI design
