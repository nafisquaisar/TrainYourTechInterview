1. Padding, Margin, Alignment

Padding 🧊

Adds inner spacing inside a widget.

Padding(
  padding: EdgeInsets.all(16),
  child: Text("Hello"),
);

Margin (via Container) 📤

Adds outer spacing around a widget.

Container(
  margin: EdgeInsets.symmetric(vertical: 10, horizontal: 20),
  child: Text("With margin"),
);

Alignment 🎯

Positions a child inside its parent.

Align(
  alignment: Alignment.centerRight,
  child: Text("Aligned Text"),
);

2. Flex, Expanded, Flexible

Expanded ➡️

Fills remaining space in Row or Column.

Row(
  children: [
    Expanded(child: Container(color: Colors.red, height: 50)),
    Container(width: 100, color: Colors.blue, height: 50),
  ],
);

Flexible 🔄

Allows flexible resizing without forcing full expansion.

Flexible(
  flex: 2,
  child: Container(color: Colors.green),
);

3. MediaQuery 📱

Used for responsive UI.

final size = MediaQuery.of(context).size;
Container(
  width: size.width * 0.5,
  height: size.height * 0.2,
);

4. SafeArea 🛡️

Prevents overlap with system UI.

SafeArea(child: Text("Inside safe area"));

5. Scroll Widgets Overview 📜

Widget Use Case
🔽 SingleChildScrollView Scroll one long child
📋 ListView Scroll list items
🟦 GridView Grid-based layout
🧩 CustomScrollView Slivers and complex scrolls
📄 PageView Page-by-page scrolling
🧭 NestedScrollView Header + scrollable body
📌 Scrollbar Visible scroll indicator

6. SingleChildScrollView 🔽

For scrolling one large child.

SingleChildScrollView(
  child: Column(children: [...]),
);

7. ListView – All Types 📋

Default ListView 

ListView(
  children: [ListTile(title: Text("Item 1"))],
);

ListView.builder ⚡

Best for long or dynamic lists.

ListView.builder(
  itemCount: 20,
  itemBuilder: (context, index) => ListTile(title: Text("Item $index")),
);

ListView.separated ➖ 

ListView.separated(
  itemCount: 10,
  itemBuilder: (context, index) => ListTile(title: Text("Item $index")),
  separatorBuilder: (_, __) => Divider(),
);

ListView.custom 🧰 

ListView.custom(
  childrenDelegate: SliverChildListDelegate([
    ListTile(title: Text("Item A")),
  ]),
);

Horizontal ListView ↔️ 

ListView(
  scrollDirection: Axis.horizontal,
  children: [Container(width: 100)],
);

8. GridView – All Types 🟦

GridView.count 🔢

Fixed number of columns.

GridView.count(crossAxisCount: 2, children: [...]);

GridView.builder ⚙️

Efficient for large lists.

GridView.builder(
  gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(crossAxisCount: 3),
  itemBuilder: (_, i) => Container(color: Colors.blue),
);

GridView.extent 📏

Max width of each tile.

GridView.extent(maxCrossAxisExtent: 150, children: [...]);

GridView.custom 🧩

Custom layout via slivers.

GridView.custom(
  gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(crossAxisCount: 2),
  childrenDelegate: SliverChildListDelegate([...]),
);

9. CustomScrollView 🧩

Used for slivers.

CustomScrollView(
  slivers: [
    SliverAppBar(expandedHeight: 200),
    SliverList(delegate: SliverChildBuilderDelegate(...)),
  ],
);

10. PageView 📄

Page-based scrolling.

PageView(children: [Container(color: Colors.red)]);

11. NestedScrollView 🧭

Header + scrollable body.

NestedScrollView(
  headerSliverBuilder: (_, __) => [SliverAppBar(expandedHeight: 200)],
  body: ListView.builder(itemCount: 30, itemBuilder: (_, i) => Text("Item $i")),
);

12. Scrollbar 📌 

Scrollbar(
  child: ListView.builder(itemCount: 30, itemBuilder: (_, i) => Text("Item $i")),
);

13. Wrap Widget 🏷️

Responsive layout with auto line breaks.

Wrap(
  spacing: 10,
  runSpacing: 10,
  children: [...],
);

Main Properties

  • ➡️ direction
  • 🎯 alignment
  • ↔️ spacing
  • ↕️ runSpacing
  • 📏 runAlignment
  • 📐 crossAxisAlignment
  • ✂️ clipBehavior

14. SizedBox 📏

Used for spacing or fixed sizes.

SizedBox(height: 20);
SizedBox(width: 100, child: Text("Button"));

Extra Features

  • SizedBox.shrink() – 🪶 zero size
  • SizedBox.expand() – 🔄 fills space

15. Custom Widgets 🧱

Reusable UI components.

class CustomCard extends StatelessWidget {
  final String title;
  CustomCard(this.title);

  Widget build(BuildContext context) {
    return Container(
      padding: EdgeInsets.all(16),
      margin: EdgeInsets.all(8),
      decoration: BoxDecoration(
        color: Colors.grey[200],
        borderRadius: BorderRadius.circular(12),
      ),
      child: Text(title),
    );
  }
}
