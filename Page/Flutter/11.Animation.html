11. Animations in Flutter

Animation in Flutter means changing a widget’s properties over time such as size, position, opacity, color, or rotation.

Flutter provides two main approaches to animations:

  • ✨ Implicit animations (simple, less control)
  • ⚙️ Explicit animations (powerful, full control)

1. Implicit Animations

What are Implicit Animations?

Implicit animations automatically animate when a widget property changes. You don’t need to manage animation controllers.

Key Points

  • ✅ Easy to use
  • 🧩 Less code
  • 🔄 Triggered using setState
  • 🎯 Best for simple UI animations

Common Implicit Animation Widgets

  • 📦 AnimatedContainer
  • 🌫️ AnimatedOpacity
  • 📐 AnimatedPadding
  • 🎯 AnimatedAlign
  • 📍 AnimatedPositioned
  • 🔁 AnimatedSwitcher
  • 🎯 AnimatedCrosFade

1️⃣ AnimatedContainer

What it animates

  • Height
  • Width
  • Color
  • Padding
  • Margin
  • Alignment
  • Decoration

Example 

double width = 100;
double height = 100;
Color color = Colors.pink;

AnimatedContainer(
  duration: Duration(seconds: 1),
  curve: Curves.easeInOut,
  width: width,
  height: height,
  color: color,
);

Control using setState 

setState(() {
  width = 200;
  height = 150;
  color = Colors.green;
});

2️⃣ AnimatedOpacity

What it animates

  • Opacity (0.0 → 1.0)

Example 

double opacity = 1.0;

AnimatedOpacity(
  opacity: opacity,
  duration: Duration(seconds: 1),
  child: Container(
    height: 100,
    width: 200,
    color: Colors.blue,
  ),
);

Control 

setState(() {
  opacity = opacity == 1 ? 0 : 1;
});

AnimatedPadding(

What it animates

  • top, left, right, bottom

Example 

bool moved = false;

Stack(
  children: [
    AnimatedPositioned(

  moved = !moved;

6️⃣ AnimatedSwitcher

What it animates

  • Widget replacement

Example 

bool showFirst = true;

AnimatedSwitcher(
  duration: Duration(seconds: 1),
  child: showFirst
      ? Container(
          key: ValueKey(1),
          height: 80,
          width: 80,
          color: Colors.green,
        )
      : Container(
          key: ValueKey(2),
          height: 80,
          width: 80,
          color: Colors.blue,
        ),
);

Control 

setState(() {
  showFirst = !showFirst;
});

7️⃣ AnimatedCrossFade

What it animates

  • Cross-fade between two widgets (fade + size)

Example 

bool showFirst = true;

AnimatedCrossFade(
  firstChild: Container(
    height: 100,
    width: 200,
    color: Colors.pink,
    child: Center(child: Text("First")),
  ),
  secondChild: Container(
    height: 150,
    width: 200,
    color: Colors.green,
    child: Center(child: Text("Second")),
  ),
  crossFadeState: showFirst
      ? CrossFadeState.showFirst
      : CrossFadeState.showSecond,
  duration: Duration(seconds: 1),
);

Control 

setState(() {
  showFirst = !showFirst;
});

🔁 How You Control Implicit Animations (Your Pattern)

All implicit animations follow the same rule:

1. Create a variable
2. Use it in AnimatedWidget
3. Change it inside setState()

Example Pattern 

setState(() {
  width = 200;
  height = 100;
  opacity = 0.0;
  color = Colors.green;
});

🧠 Key Interview Lines

  • Implicit animations run automatically on value change
  • No AnimationController required
  • Best for simple UI effects
  • Controlled using setState()

✅ When to Use Implicit Animations

  • Button click animations
  • Simple UI transitions
  • Small visual feedback
  • Splash or onboarding screens

2. Explicit Animations

What are Explicit Animations?

Explicit animations give full control over the animation using an AnimationController.

Key Points

  • 🎛️ More control
  • 🧱 More code
  • 🧠 Used for complex animations
  • ▶️ Start, stop, repeat, reverse manually

Common Widgets

  • 🏗️ AnimatedBuilder
  • 🌫️ FadeTransition
  • 🔍 ScaleTransition
  • 🔄 RotationTransition
  • ➡️ SlideTransition

3. AnimationController

What is AnimationController?

AnimationController manages the duration and lifecycle of an animation.

Important Points

  • ⏱️ Requires TickerProvider
  • 🧩 Used inside StatefulWidget
  • 🧹 Must be disposed to avoid memory leaks

Example 

class MyAnim extends StatefulWidget {
  @override
  _MyAnimState createState() => _MyAnimState();
}

class _MyAnimState extends State<MyAnim>
    with SingleTickerProviderStateMixin {

  late AnimationController controller;

  @override
  void initState() {
    super.initState();
    controller = AnimationController(
      vsync: this,
      duration: Duration(seconds: 2),
    );
    controller.forward();
  }

  @override
  void dispose() {
    controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Container();
  }
}

4. Tween and Curves

Tween

What is a Tween?

Tween stands for in-between. It defines the start and end values of an animation.

Tween<double>(begin: 0.0, end: 1.0)

A Tween does not animate by itself. It needs an AnimationController to control time.

How Tween Animation Works 

AnimationController  → controls TIME
Tween                → controls VALUES
Animation             → combines both

Basic Tween Example (Double Tween) 

Animation<double> animation =
  Tween<double>(begin: 0.0, end: 200.0)
      .animate(controller);

This means:

  • ▶️ Animation starts at 0.0
  • 🏁 Animation ends at 200.0
  • ⏱️ Speed is controlled by the controller

Types of Tween in Flutter

1. DoubleTween

Used for:

  • 📏 Size
  • 🌫️ Opacity
  • 🔄 Rotation
  • 🔍 Scale
Tween<double>(begin: 0.0, end: 1.0)

2. ColorTween

ColorTween is used to animate between two colors.

ColorTween(
  begin: Colors.red,
  end: Colors.blue,
)

Usage:

Animation<Color?> colorAnimation =
  ColorTween(
    begin: Colors.red,
    end: Colors.green,
  ).animate(controller);

3. SizeTween

Used to animate width and height together.

SizeTween(
  begin: Size(50, 50),
  end: Size(150, 150),
)

4. OffsetTween

Used for sliding animations.

Tween<Offset>(
  begin: Offset(0, 1),
  end: Offset(0, 0),
)

5. IntTween

Used for number or counter animations.

IntTween(begin: 0, end: 100)

6. RectTween

Used in Hero and page transition animations.

What is AnimationController?

AnimationController controls the time and playback of an animation.

It manages:

  • ⏳ Duration
  • ▶️ Start and stop
  • 🔁 Repeat and reverse
AnimationController(
  vsync: this,
  duration: Duration(seconds: 2),
);

Common Controller Methods 

controller.forward();   // ▶️ start animation
controller.reverse();   // ◀️ reverse animation
controller.repeat();    // 🔁 repeat animation
controller.stop();      // ⏹️ stop animation

What is SingleTickerProviderStateMixin?

Animations need a Ticker to sync with screen refresh (~60 FPS).

SingleTickerProviderStateMixin:

  • 🎯 Provides one ticker
  • ☝️ Used when you have one AnimationController
class MyWidgetState extends State<MyWidget>
    with SingleTickerProviderStateMixin {

If you use multiple controllers, use:

with TickerProviderStateMixin

Complete Example (Tween + ColorTween + Controller) 

import 'package:flutter/material.dart';

class TweenDemo extends StatefulWidget {
  @override
  State<TweenDemo> createState() => _TweenDemoState();
}

class _TweenDemoState extends State<TweenDemo>
    with SingleTickerProviderStateMixin {

  late AnimationController controller;
  late Animation<double> sizeAnimation;
  late Animation<Color?> colorAnimation;

  @override
  void initState() {
    super.initState();

    controller = AnimationController(
      vsync: this,
      duration: Duration(seconds: 2),
    );

    sizeAnimation = Tween<double>(
      begin: 100,
      end: 200,
    ).animate(CurvedAnimation(
      parent: controller,
      curve: Curves.easeInOut,
    ));

    colorAnimation = ColorTween(
      begin: Colors.pink,
      end: Colors.green,
    ).animate(controller);

    controller.repeat(reverse: true);
  }

  @override
  void dispose() {
    controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text("Tween Animation")),
      body: Center(
        child: AnimatedBuilder(
          animation: controller,
          builder: (context, child) {
            return Container(
              width: sizeAnimation.value,
              height: sizeAnimation.value,
              color: colorAnimation.value,
            );
          },
        ),
      ),
    );
  }
}

Hero animation creates a smooth transition of a widget between two screens.

Use Cases

Ripple Animation in Flutter

What is Ripple Animation?

Ripple animation is a wave-like circular animation that expands and fades, similar to water ripples.

It is commonly used for:

  • 📞 Call or notification effects
  • 🚨 Attention indicators
  • 👆 Touch feedback visuals
  • ⏳ Loading or alert animations

How Ripple Animation Works

Ripple animation is built using:

  • 🎛️ AnimationController → controls time
  • 📐 Tween → controls size progression
  • 🔄 AnimatedBuilder → rebuilds UI on every frame
  • 🧱 Stack + Container → draws expanding circles

Core Logic (Concept) 

AnimationController → changes value (0 → 1)
Tween               → maps value to scale
Container           → grows in size
Opacity             → decreases with time

Simple Ripple Animation Example 

class RippleAnimation extends StatefulWidget {
  @override
  State<RippleAnimation> createState() => _RippleAnimationState();
}

class _RippleAnimationState extends State<RippleAnimation>
    with SingleTickerProviderStateMixin {

  late AnimationController controller;
  late Animation<double> animation;

  final radiusList = [50, 100, 150, 200];

  @override
  void initState() {
    super.initState();

    controller = AnimationController(
      vsync: this,
      duration: Duration(seconds: 1),
    );

    animation = Tween<double>(begin: 0.0, end: 1.0).animate(controller);

    controller.repeat();
  }

  @override
  void dispose() {
    controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: AnimatedBuilder(
          animation: controller,
          builder: (context, child) {
            return Stack(
              alignment: Alignment.center,
              children: radiusList.map((radius) {
                double scale = 0.3 + animation.value * 0.7;

                return Container(
                  width: radius * scale,
                  height: radius * scale,
                  decoration: BoxDecoration(
                    shape: BoxShape.circle,
                    color: Colors.teal.withOpacity(
                      (1 - animation.value).clamp(0.2, 1.0),
                    ),
                  ),
                );
              }).toList(),
            );
          },
        ),
      ),
    );
  }
}

Important Points

  • 🔄 AnimatedBuilder is required to update UI
  • 🎨 Opacity should be clamped to avoid full disappearance
  • 📏 Minimum scale prevents size becoming zero
  • 🌀 Stack is used to create multiple ripple waves

Common Mistakes

  • ❌ Forgetting to use AnimatedBuilder
  • ❌ Allowing ripple size to reach zero
  • ❌ Not clamping opacity values
  • ❌ Not disposing AnimationController
