🤖 Kotlin Interview Questions (Android-Focused)

📘 What is Kotlin?

Kotlin is a modern programming language developed by JetBrains, mainly used for Android application development and fully compatible with Java.

⭐ Why is Kotlin preferred over Java for Android?

Kotlin is preferred because it has less boilerplate code, built-in null safety, better readability, and reduces common errors like NullPointerException.

🔁 Difference between val and var

val is immutable and cannot be reassigned, while var is mutable and can be reassigned.

🛡️ What is null safety in Kotlin?

Null safety is a Kotlin feature that helps prevent NullPointerException by forcing developers to handle null values explicitly.

var name: String = "Nafis"
// name = null  ❌ Compile-time error

//If you want to allow null:
var name: String? = "Nafis"
name = null  // ✅ Allowed

//To safely use a nullable variable:
var name: String? = "Nafis"
println(name?.length)

❓ What is ? and !! in Kotlin?

? allows safe handling of nullable values, while !! forcefully converts a nullable value to non-null and throws NullPointerException if the value is null.

1. Safe call operator ?.

  • Used when a variable may be null.
    If the value is null, it returns null instead of crashing.
var name: String? = "Nafis"
println(name?.length)   // Output: 5

If name is null:
name = null
println(name?.length)   // Output: null (no crash)

👉 Use this when you want safe access.

2. Elvis operator ?:

  • Used to provide a default value when something is null.
var name: String? = null
val length = name?.length ?: 0
println(length)   // Output: 0

👉 Meaning:

“If name?.length is null, use 0.”

3. Not-null assertion operator !!

  • Forces Kotlin to treat a variable as non-null.
    If the value is actually null, the app crashes.
var name: String? = "Nafis"
println(name!!.length)   // Output: 5

If name is null:
name = null
println(name!!.length)   // ❌ NullPointerException

📦 What are data classes?

Data classes are used to store data and automatically generate methods like toString(), equals(), hashCode(), and copy().

🧩 What is a companion object?

A companion object is used to define static-like members in Kotlin that can be accessed using the class name.

⏳ Difference between lateinit and lazy

lateinit is used for variables initialized later and works only with var, while lazy initializes values on first access and works with val.

🧱 What is a sealed class?

A sealed class represents a restricted class hierarchy and is commonly used to handle states like success, error, and loading.

🔀 Difference between sealed class and enum

Enums hold fixed constant values, while sealed classes can hold objects with different data and are more flexible.

🔄 What is when expression?

when is used as a replacement for switch and handles multiple conditions in a clean and readable way.

⚖️ Difference between == and === in Kotlin

== checks structural equality (value comparison),

while === checks referential equality (memory reference).

➕ What are extension functions?

Extension functions allow adding new functions to existing classes without modifying their source code.

🧵 What is coroutine?

Coroutines are used to perform asynchronous and background tasks efficiently without blocking the main thread.

⏸️ Difference between suspend function and normal function

A suspend function can pause and resume execution inside coroutines, while a normal function runs synchronously.

🎯 What are scope functions (let, apply, also, run, with)?

Scope functions are used to work with an object inside a block, so the code becomes shorter and cleaner.

Quick use:

  • 🔹 let → 👉 Used when a value may be null
name?.let {
    println(it)
}
  • 🔹 apply → apply is used to configure or initialize an object. It lets you set multiple properties of an object and returns the same object.
val user = User().apply {
    name = "Nafis"
    age = 22
}
  • 🔹 also → 👉 also is used to perform an additional action on an object and it returns the same object.
list.also {
    println(it.size)
}
  • 🔹 run / with → run is a scope function used to execute a block of code on an object and return the result of that block.
val length = name.run {
    length
}


with(user) {
    println(name)
    println(age)
}

🧠 What is smart cast?

Smart cast automatically converts a variable to a specific type after checking its type or null condition.

🚫 How Kotlin handles NullPointerException?

Kotlin prevents NullPointerException using null safety features like nullable types, safe calls (?.), and the Elvis operator (?:).

⚔️ Java vs Kotlin

🆚 Difference between Java and Kotlin

Java has more boilerplate code and lacks null safety, while Kotlin is concise, null-safe, and designed for modern Android development.

✂️ Why Kotlin code is shorter than Java?

Kotlin provides features like data classes and type inference that significantly reduce code length.

🔗 Is Kotlin fully compatible with Java?

Yes, Kotlin is fully interoperable with Java.

🤝 Can we use Java and Kotlin together in Android?

Yes, Java and Kotlin can be used together within the same Android project.

🔥 Interview-Ready Closing Line

I use Java for strong core concepts and DSA, and Kotlin for Android development because it is concise and null-safe.
