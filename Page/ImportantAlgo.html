This note is written in interview-speaking style. You can explain directly from this.

1️⃣ Dijkstra’s Algorithm

Short Definition

Dijkstra’s algorithm finds the shortest path from a single source to all other vertices in a graph with non‑negative edge weights.

When to Use

  • When graph has non-negative edge weights
  • When you need fast shortest path from one source

Interview Explanation

  • Greedy algorithm
  • Picks the nearest unvisited node every time
  • Distance never decreases once finalized

Key Conditions

  • ❌ Negative edges not allowed
  • ❌ Cannot detect negative cycle

Time & Space

  • Time: O(E log V) using min-heap
  • Space: O(V)

Java Code (Using Priority Queue) 

import java.util.*;

class Dijkstra {
    static class Pair {
        int node, dist;
        Pair(int n, int d) { node = n; dist = d; }
    }

    public static void dijkstra(int V, List<List<Pair>> graph, int src) {
        int[] dist = new int[V];
        Arrays.fill(dist, Integer.MAX_VALUE);
        dist[src] = 0;

        PriorityQueue<Pair> pq = new PriorityQueue<>(Comparator.comparingInt(a -> a.dist));
        pq.add(new Pair(src, 0));

        while (!pq.isEmpty()) {
            Pair cur = pq.poll();
            int u = cur.node;

            for (Pair nbr : graph.get(u)) {
                int v = nbr.node;
                int wt = nbr.dist;

                if (dist[u] + wt < dist[v]) {
                    dist[v] = dist[u] + wt;
                    pq.add(new Pair(v, dist[v]));
                }
            }
        }

        System.out.println(Arrays.toString(dist));
    }
}

2️⃣ Bellman–Ford Algorithm

Short Definition

Bellman–Ford algorithm finds the shortest path from a single source in a graph that may contain negative edge weights and can also detect negative cycles.

When to Use

  • When graph has negative edges
  • When you must detect negative cycles

Interview Explanation

  • Relaxes all edges V − 1 times
  • One extra relaxation checks for negative cycle

Key Conditions

  • ✅ Handles negative weight edges
  • ✅ Detects negative weight cycle
  • ❌ Slower than Dijkstra

Time & Space

  • Time: O(V × E)
  • Space: O(V)

Java Code 

class BellmanFord {
    static class Edge {
        int u, v, w;
        Edge(int u, int v, int w) {
            this.u = u; this.v = v; this.w = w;
        }
    }

    static void bellmanFord(int V, int src, List<Edge> edges) {
        int[] dist = new int[V];
        Arrays.fill(dist, Integer.MAX_VALUE);
        dist[src] = 0;

        for (int i = 1; i < V; i++) {
            for (Edge e : edges) {
                if (dist[e.u] != Integer.MAX_VALUE && dist[e.u] + e.w < dist[e.v]) {
                    dist[e.v] = dist[e.u] + e.w;
                }
            }
        }

        for (Edge e : edges) {
            if (dist[e.u] != Integer.MAX_VALUE && dist[e.u] + e.w < dist[e.v]) {
                System.out.println("Negative cycle detected");
                return;
            }
        }

        System.out.println(Arrays.toString(dist));
    }
}

3️⃣ Floyd–Warshall Algorithm

Short Definition

Floyd–Warshall algorithm finds the shortest paths between all pairs of vertices in a weighted graph using dynamic programming.

When to Use

  • When you need shortest path between all pairs
  • Graph size is small

Interview Explanation

  • Uses dynamic programming
  • Tries every node as an intermediate path

Key Conditions

  • ✅ Handles negative weights
  • ❌ Cannot handle negative cycle
  • ❌ High time complexity

Time & Space

  • Time: O(V³)
  • Space: O(V²)

Java Code 

class FloydWarshall {
    static final int INF = 99999;

    static void floydWarshall(int[][] graph, int V) {
        int[][] dist = new int[V][V];

        for (int i = 0; i < V; i++)
            for (int j = 0; j < V; j++)
                dist[i][j] = graph[i][j];

        for (int k = 0; k < V; k++) {
            for (int i = 0; i < V; i++) {
                for (int j = 0; j < V; j++) {
                    if (dist[i][k] + dist[k][j] < dist[i][j])
                        dist[i][j] = dist[i][k] + dist[k][j];
                }
            }
        }

        for (int i = 0; i < V; i++)
            System.out.println(Arrays.toString(dist[i]));
    }
}

👑 N-Queen Problem (Interview Notes)

📌 Definition

The N-Queen Problem is a classic backtracking problem where the goal is to place N queens on an N×N chessboard such that no two queens attack each other.

A queen can attack another queen if they are on:

  • The same row
  • The same column
  • The same diagonal

🧠 Algorithm (Backtracking Approach)

We solve the problem using backtracking, placing one queen at a time and checking whether it is safe.

Steps

  1. Start with an empty N×N board

  2. Try to place a queen in the first column

  3. For each row in that column:

    • Check if placing a queen is safe
    • If safe, place the queen

  4. Move to the next column and repeat

  5. If no position is safe, backtrack (remove the last queen)

  6. Continue until all queens are placed

✔️ Safe Position Check

Before placing a queen at position (row, col), ensure:

  • No queen in the same row (left side)
  • No queen in the upper-left diagonal
  • No queen in the lower-left diagonal

💻 Java Code (Backtracking) 

class NQueen {

    static void solveNQueen(int n) {
        int[][] board = new int[n][n];
        placeQueen(board, 0, n);
    }

    static boolean placeQueen(int[][] board, int col, int n) {
        if (col >= n) {
            printBoard(board);
            return true; // for one solution
        }

        for (int row = 0; row < n; row++) {
            if (isSafe(board, row, col, n)) {
                board[row][col] = 1;

                if (placeQueen(board, col + 1, n))
                    return true;

                board[row][col] = 0; // backtrack
            }
        }
        return false;
    }

    static boolean isSafe(int[][] board, int row, int col, int n) {
        // Check left row
        for (int i = 0; i < col; i++)
            if (board[row][i] == 1)
                return false;

        // Check upper-left diagonal
        for (int i = row, j = col; i >= 0 && j >= 0; i--, j--)
            if (board[i][j] == 1)
                return false;

        // Check lower-left diagonal
        for (int i = row, j = col; i < n && j >= 0; i++, j--)
            if (board[i][j] == 1)
                return false;

        return true;
    }

    static void printBoard(int[][] board) {
        for (int[] row : board) {
            for (int cell : row)
                System.out.print(cell + " ");
            System.out.println();
        }
        System.out.println();
    }

    public static void main(String[] args) {
        solveNQueen(4);
    }
}

⏱ Time Complexity

  • Worst-case: O(N!)
  • This happens because we try all possible positions using backtracking
  • Space Complexity: O(N²) (for board storage)

🎯 Interview One-Line Answer

"The N-Queen problem is solved using backtracking by placing queens one by one in safe positions and backtracking when a conflict occurs."

🧠 Important Algorithm Concepts

🔹 1️⃣ 0/1 Knapsack Problem

Definition (How to Say in Interview)

The 0/1 Knapsack problem is a dynamic programming problem where each item can be either taken once or not taken at all. The goal is to maximize total value without exceeding the weight limit.

Core Idea

  • For every item, we have two choices:

    • Include the item
    • Exclude the item

  • We choose the option that gives maximum value

Why Dynamic Programming?

Because the same subproblems are solved repeatedly.

Time & Space Complexity

  • Time: O(n × W)
  • Space: O(n × W)

Java Code 

class Knapsack01 {
    static int knapsack(int[] wt, int[] val, int W, int n) {
        int[][] dp = new int[n + 1][W + 1];

        for (int i = 0; i <= n; i++) {
            for (int w = 0; w <= W; w++) {
                if (i == 0 || w == 0)
                    dp[i][w] = 0;
                else if (wt[i - 1] <= w)
                    dp[i][w] = Math.max(val[i - 1] + dp[i - 1][w - wt[i - 1]], dp[i - 1][w]);
                else
                    dp[i][w] = dp[i - 1][w];
            }
        }
        return dp[n][W];
    }
}

🔹 2️⃣ Longest Common Subsequence (LCS)

Definition (Interview Friendly)

The Longest Common Subsequence is the longest sequence of characters that appears in both strings in the same order, but not necessarily continuously.

Core Idea

  • Compare characters of two strings
  • If characters match → include
  • If not → take max of two possibilities

Example

  • String 1: ABCD
  • String 2: ACBD
  • LCS: ABD

Time & Space Complexity

  • Time: O(n × m)
  • Space: O(n × m)

Java Code 

class LCS {
    static int lcs(String s1, String s2) {
        int n = s1.length(), m = s2.length();
        int[][] dp = new int[n + 1][m + 1];

        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {
                if (s1.charAt(i - 1) == s2.charAt(j - 1))
                    dp[i][j] = 1 + dp[i - 1][j - 1];
                else
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
        return dp[n][m];
    }
}

🔹 3️⃣ Infix to Postfix Conversion

Definition

Infix expression places operator between operands, while postfix places operator after operands.

Interview Explanation

We use a stack to maintain operators based on precedence.

Core Steps

  • If operand → output directly
  • If operator → pop higher/equal precedence operators
  • If '(' → push
  • If ')' → pop until '('

Java Code 

import java.util.*;

class InfixToPostfix {
    static int prec(char c) {
        if (c == '+' || c == '-') return 1;
        if (c == '*' || c == '/') return 2;
        return -1;
    }

    static String convert(String exp) {
        StringBuilder res = new StringBuilder();
        Stack<Character> st = new Stack<>();

        for (char c : exp.toCharArray()) {
            if (Character.isLetterOrDigit(c))
                res.append(c);
            else if (c == '(')
                st.push(c);
            else if (c == ')') {
                while (!st.isEmpty() && st.peek() != '(')
                    res.append(st.pop());
                st.pop();
            } else {
                while (!st.isEmpty() && prec(c) <= prec(st.peek()))
                    res.append(st.pop());
                st.push(c);
            }
        }
        while (!st.isEmpty()) res.append(st.pop());
        return res.toString();
    }
}

🔹 4️⃣ Balanced Parentheses

Definition

Parentheses are balanced if every opening bracket has a matching closing bracket in correct order.

Interview Explanation

We use a stack to track opening brackets.

Java Code 

import java.util.*;

class BalancedParentheses {
    static boolean isBalanced(String s) {
        Stack<Character> st = new Stack<>();

        for (char c : s.toCharArray()) {
            if (c == '(' || c == '{' || c == '[')
                st.push(c);
            else {
                if (st.isEmpty()) return false;
                char top = st.pop();
                if ((c == ')' && top != '(') ||
                    (c == '}' && top != '{') ||
                    (c == ']' && top != '['))
                    return false;
            }
        }
        return st.isEmpty();
    }
}

🔹 5️⃣ Rat in a Maze

Definition

Rat in a Maze is a backtracking problem where a rat must find a path from source to destination.

Interview Explanation

We try all possible paths and backtrack when a path is blocked.

Java Code 

class RatMaze {
    static boolean solve(int[][] maze, int x, int y, int n, int[][] sol) {
        if (x == n - 1 && y == n - 1 && maze[x][y] == 1) {
            sol[x][y] = 1;
            return true;
        }
        if (x >= 0 && y >= 0 && x < n && y < n && maze[x][y] == 1) {
            sol[x][y] = 1;
            if (solve(maze, x + 1, y, n, sol)) return true;
            if (solve(maze, x, y + 1, n, sol)) return true;
            sol[x][y] = 0;
        }
        return false;
    }
}

🔹 6️⃣ Subsets and Permutations

Subsets (Power Set)

Idea: Include or exclude each element.

class Subsets {
    static void subsets(int[] arr, int idx, String cur) {
        if (idx == arr.length) {
            System.out.println(cur);
            return;
        }
        subsets(arr, idx + 1, cur);
        subsets(arr, idx + 1, cur + arr[idx]);
    }
}

Permutations

Idea: Fix one element and permute the rest.

class Permutations {
    static void permute(String s, String ans) {
        if (s.length() == 0) {
            System.out.println(ans);
            return;
        }
        for (int i = 0; i < s.length(); i++) {
            permute(s.substring(0, i) + s.substring(i + 1), ans + s.charAt(i));
        }
    }
}
